# Material 3

## Patterns

- Use `ColorScheme` and `TextTheme` as the base; avoid hard-coded colors.
- Prefer components that adapt well to large screens (navigation rail, drawers).

## Pitfalls

- Mixing custom colors with `ColorScheme` without semantic mapping.
