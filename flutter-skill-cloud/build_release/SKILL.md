# Build & Release Skills

## What this domain covers

Flavors, signing, CI/CD concepts, versioning, and store release considerations.

## When to read it

- You’re adding staging vs prod environments.
- You’re preparing a release and need reproducible builds.

## Subtopics

- Flavors: [flavors.md](./flavors.md)
- Signing: [signing.md](./signing.md)
- CI/CD: [ci_cd.md](./ci_cd.md)
- Versioning: [versioning.md](./versioning.md)
- App Store: [appstore.md](./appstore.md)
- Play Store: [playstore.md](./playstore.md)

## Decision guide

- If you need **multiple environments**, go to [flavors.md](./flavors.md).
- If you need **reproducible releases**, go to [signing.md](./signing.md) and [versioning.md](./versioning.md).
